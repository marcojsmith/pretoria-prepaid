# Project Brief

## Overview

**Project Name:** Pretoria Prepaid  
**Type:** Web Application (Digital Platform)

Pretoria Prepaid is a digital platform for managing prepaid electricity tokens. The application enables users to manage their prepaid electricity accounts, record purchased tokens, view transaction history, and monitor their electricity usage.

## Purpose

The platform provides a convenient way for residents in Pretoria to:

- Purchase prepaid electricity tokens
- View their account balance and transaction history
- Monitor electricity rates and pricing
- Manage their prepaid electricity accounts

## Target Audience

- Residents in Pretoria area with prepaid electricity meters
- Users who prefer digital payment methods over physical token purchases

## Key Features

- User authentication and account management
- Recording of prepaid token purchases
- Transaction history and receipt viewing
- Real-time electricity rate information
- Responsive web interface for mobile and desktop

## Technology Stack

- **Frontend:** React (Vite), TypeScript
- **Backend/Database:** Convex
- **Authentication:** Clerk
- **UI Components:** shadcn/ui (Radix UI)
- **Styling:** Tailwind CSS

## Goals

- Provide a seamless platform for recording and tracking prepaid token purchases
- Ensure real-time synchronization of account data
- Deliver a responsive, user-friendly experience across devices
- Maintain high code quality and security standards
